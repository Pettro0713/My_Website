Wikipedia

The goal of this project to scrape data from the Wikipedia Home page and parse it through various web scraping techniques. Got familiar with various web scraping techniques, Java modules for web scraping, and processes of Data extraction and data processing. Now, we don’t have to manually copy and paste data from wikipedia but this scraper can perform that task for us in a couple of seconds.

Interactive software application that pulls live data from Wikipedia about most recent revisions from a given Wikipedia article.
